Package Delivery Problem

  • Given a graph with n nodes and m edges, where each edge has a weight associated with it.
  • Each node represents a house and each edge represents a road connecting two houses.
  • Each edge has a weight associated with it which represents the distance between the two houses.
  • There is a delivery agent who has to deliver a set of packages to diffenet locations(houses).
  • The agent has to find the most optimal path to deliver the said packages to
  • The program will also figure out the most optimal order in which to deliver packages so as to get the most optimal delivery path.
